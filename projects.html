433 MHz Wireless RF Control System for RC Car

Custom RF transmitter/receiver chain with QPSK modulation implemented end-to-end in hardware and embedded software.

RFQPSKPSoC 6Analog DesignADS
+
Add photo: <img src="rfcar-board.jpg">
Add photo: <img src="rfcar-schematic.jpg">

Overview

This senior capstone project involves designing a complete wireless RF control system operating at 433 MHz. The goal is to build the full signal chain from baseband generation through to motor actuation — avoiding off-the-shelf RF modules in favor of a from-scratch hardware implementation.

Transmitter

  • DAC-based baseband QPSK signal generation on PSoC 6
  • Analog filtering and upconversion via discrete mixer stages to 433 MHz
  • Amplification and impedance matching to antenna

Receiver

  • Low-noise amplification, bandpass filtering, and downconversion
  • QPSK demodulation and timing recovery implemented in embedded software
  • Real-time decoded control output to motor driver and servo subsystems

Embedded System

  • PSoC 6 handles baseband generation, timing recovery, and actuator control
  • Integrated with motor control, steering servo, and onboard sensors
  • Custom packet framing and error detection scheme

8-Channel EEG Brain-Computer Interface

Custom EEG acquisition PCB with analog amplification and filtering stages targeting neural signal capture for movement intention detection.

PCB DesignBiomedicalDSPRaspberry PiAltium
+
Add photo: <img src="bci-pcb-v1.jpg">
Add photo: <img src="bci-signal-capture.png">

Overview

A personal long-running project to build a non-invasive brain-computer interface from scratch. The first hardware revision used instrumentation amplifier ICs; the current redesign targets custom chopper instrumentation amplifiers to reduce pink noise and improve signal integrity while shrinking board area and cost.

Hardware — PCB Design

  • 8-channel EEG acquisition with per-channel analog amplification and band-pass filtering
  • High-resolution ADC communicating to Raspberry Pi 4B via SPI
  • Current redesign: replacing IA ICs with custom chopper IA topology to improve 1/f noise
  • Compact, battery-powered form factor with EMI shielding considerations

Signal Processing & Software

  • SPI-based data acquisition pipeline on Raspberry Pi 4B
  • FIR notch and bandpass filters for powerline rejection and alpha/beta/theta band isolation
  • Exploring wavelet transform decomposition and ML-based movement intention classification

3D Simulation Interface

  • Built a 3D avatar control simulation in Unity/C# demonstrating the end-to-end system concept
  • User currently controls avatar via mouse; to be replaced with classified neural signals

Personal Project

High Voltage SCR AC Power Dimmer

Silicon Controlled Rectifier based AC dimmer controlling power flow in high-voltage systems by varying the thyristor firing angle.

Power ElectronicsSCRAC ControlPhase Control
+

Overview

Designed, assembled, and tested a high-voltage SCR-based AC dimmer circuit. The system controls power delivered to a resistive load by varying the firing angle of the silicon controlled rectifier, adjusting the point in each AC half-cycle at which current begins to flow.

Key Design Elements

  • Zero-crossing detection circuit for synchronizing firing angle to AC mains phase
  • RC phase-shift network and DIAC trigger for gate firing control
  • Snubber network for dV/dt protection and EMI reduction
  • Isolation and safety design for line-voltage operation
